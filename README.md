# Admin App

React based admin template

### Installing

Follow below steps to get ready!

Make a copy from `.env.example` to `.env`

```
cp .env.example .env
```

Install node modules

```
yarn
```

Run dev mode

```
yarn start
```

And that's it :shipit:

## Deployment

Modify your `.env` variables.

```
NODE_ENV=production
```

Build.

```
yarn build
```
